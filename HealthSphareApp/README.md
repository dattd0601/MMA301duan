# HealthSphareApp - Your Personal Health & Fitness Companion

HealthSphareApp is an Expo-based mobile application designed to help users track their health metrics, calculate nutritional needs, and monitor fitness progress.

## 🚀 Features

- **Health Metrics Calculator**: BMI, Ideal Body Weight, BMR, and Body Fat Percentage.
- **Nutritional Tracking**: Daily calorie needs (TDEE) and Macronutrient distribution.
- **Hydration Tracker**: Recommended water intake based on activity level.
- **Activity Monitoring**: Integrated support for various activity levels and goals.
- **Premium Features**: Exclusive banners and features for premium users.

## 🛠 Tech Stack

- **Framework**: [React Native](https://reactnative.dev/) with [Expo](https://expo.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Package Manager**: [Bun](https://bun.sh/) (or NPM/Yarn)
- **State Management**: Context API / Integrated Store
- **UI Components**: Custom reusable components

## 📦 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (LTS)
- [Bun](https://bun.sh/) (recommended) or NPM

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd HealthSphareApp
   ```

2. Install dependencies:
   ```bash
   bun install
   # or
   npm install
   ```

3. Start the development server:
   ```bash
   npx expo start --web
   ```
